# Marian Sandur CV

Dark editorial CV landing page built with Next.js, Space Grotesk, and JetBrains Mono.

## Commands

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

## Stack

- Next.js App Router
- Static rendering
- `next/font` for optimized typography
- Responsive dark CV layout inspired by `CV Noir.dc.html`
